# Spec-Driven Artificial Intelligence for Empirical Research - Replication Package

Replication package for the scoping-review article written under the **FSAIM** methodology (spec-driven AI for empirical research). It contains the source data, appendix materials, and a procedure description - everything needed to reproduce the results.

Methodology announced in: **Wojarnik, G. (2026)**, SSRN, DOI [10.2139/ssrn.6853581](https://doi.org/10.2139/ssrn.6853581).

## Contents

| Folder / file | Description |
|---|---|
| `PROCEDURE.md` | Single spec: the research procedure in six steps |
| `data/` | Source data: screened corpus (582 works), coding sheet (69 rows; 41 works coded from full text) |
| `data/tables/` | Result tables (CSV): synthesis matrix, approaches table, RDF in economics, SDD-to-finance mapping |
| `appendix/` | Appendices A-E: search log, coding calibration, cross-model adjudication decisions, PRISMA-ScR checklist |
| `figures/` | PRISMA-ScR flow diagram, 7-layer control map, minimal SDD workflow (PL + EN, no embedded titles) |
| `manuscript/` | Reference article (PDF, PL + EN) |
| `references.bib` | Bibliography (44 entries) |
| `.zenodo.json` | Zenodo deposit metadata |
| `CITATION.cff` | Citation metadata |
| `docs/` | AI provenance log (layer L6) and a reproduction guide |

## Full texts of works

Not included here - they are copyrighted. Locate each work by its DOI in `data/corpus_screened_in.csv` and `references.bib`.
